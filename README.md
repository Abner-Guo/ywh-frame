# YwhFrame

ywh个人学习开发框架，搭建笔记：https://blog.csdn.net/qq_36956154/article/details/84036199

## 整合

- 数据连接池 Druid
- 使用MyBatisPlus作为持久层
- 集成Redis
- 实现SpringSecurity作为安全框架管理账户

## 环境

- JDK1.8
- Redis
